# Introduction

Vue Logics is a Vue Plugin to let you build your logics based on [Flowy.js](https://github.com/alyssaxuu/flowy).

## Simple Example

```vue
<VueLogics width="500px" height="700px" />
```

**Output**:

<VueLogics width="500px" height="700px" />
